# Icon Assets

Place your add-in icon files here:

- `icon-16.png` (16x16 pixels)
- `icon-32.png` (32x32 pixels)
- `icon-64.png` (64x64 pixels)
- `icon-80.png` (80x80 pixels)

## Icon Requirements

- Format: PNG
- Background: Transparent recommended
- Design: Simple, recognizable (like a link/chain icon)
- Colors: Should match your branding

## Temporary Solution

For now, the add-in will work without icons (Outlook uses default icons).
You can add proper icons later by creating PNG files with the dimensions above and placing them in this folder.

## Quick Icon Creation

Use any of these free tools:
- [Canva](https://canva.com) - Easy drag-and-drop
- [Figma](https://figma.com) - Professional design tool
- [GIMP](https://gimp.org) - Free Photoshop alternative

Or use an online icon generator with a link/chain emoji or icon.
